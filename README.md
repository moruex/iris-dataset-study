# 🌸 Iris Veri Seti Sınıflandırma Projesi

Bu projede klasik **Iris veri seti** kullanılarak çiçek türlerinin sınıflandırılması yapılmıştır.

## 📊 Veri Seti

* 150 örnek veri içerir
* 3 sınıf: Setosa, Versicolor, Virginica
* 4 özellik: çanak yaprak ve taç yaprak uzunluk/genişlikleri

---

## ⚙️ Kullanılan Modeller

* Lojistik Regresyon
* Support Vector Machine (SVC)
* Gaussian Naive Bayes

---

## 🧪 Veri Ön İşleme

* Eğitim / test ayrımı yapılmıştır
* Özellikler ölçeklendirilmiştir (StandardScaler)

---

## 📈 Model Değerlendirme

Modeller aşağıdaki metriklerle değerlendirilmiştir:

* Doğruluk (Accuracy)
* Karmaşıklık Matrisi (Confusion Matrix)
* Precision, Recall, F1-score

---

## 🎯 Sonuç

Tüm modeller Iris veri setinde yüksek başarı göstermiştir. Veri setinin basit ve dengeli yapısı nedeniyle sınıflandırma performansı yüksektir.
